# PhotoMove Bridge Support

PhotoMove Bridge helps users export selected iPhone photos and videos to a Windows PC or external drive by USB.

## Common Checks

- Confirm that photo library permission is allowed in iOS Settings.
- Create a USB export from the Move tab before connecting to Windows.
- On Windows, select the exported PhotoMoveBridgeUSBExport folder from Apple Devices or iTunes File Sharing.
- Choose the target folder on the PC or external drive before starting import.
- Confirm import verification before deleting items from the iPhone.

## Privacy

The app works locally. It does not upload photos or videos to developer servers, cloud services, analytics services, or advertising networks.
