# PhotoMove Bridge Privacy Policy

Last updated: 2026-06-16

PhotoMove Bridge is a local-only photo and video transfer utility.

## Data Collection

PhotoMove Bridge does not collect personal data, analytics, diagnostics, or tracking data.

## Photos and Videos

With the user's permission, the iOS app reads selected photos and videos from the device photo library so the user can create a local USB export folder inside the app's Documents area.

The app does not upload photos or videos to developer servers, cloud services, analytics services, advertising networks, or any other network destination.

## USB Export

Users may manually copy exported files from the iPhone to their own Windows PC or external drive using USB file sharing and the optional Windows companion app. This transfer is controlled by the user and happens outside any developer-operated server.

## Deletion

The app can request deletion of already-copied items from the iPhone photo library only after user confirmation. iOS always shows the standard system delete confirmation.

## Third-Party Services

PhotoMove Bridge does not include third-party analytics SDKs, advertising SDKs, or tracking SDKs.

## Contact

For privacy or support questions, use the support information listed in this repository.
